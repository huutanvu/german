# übertragen

## Meanings
- to transfer (data, code)
- to transmit / broadcast
- to delegate / assign (tasks)

## Grammar
- Part of speech: Verb (strong, irregular, inseparable)
- Perfekt: hat übertragen
- Conjugation:
  - ich übertrage
  - du überträgst
  - er/sie/es überträgt
  - wir übertragen
  - ihr übertragt
  - sie/Sie übertragen
  - Past tense (Präteritum): übertrug

## Daily Use Case
- Das Fußballspiel wird live im Fernsehen übertragen.

## Professional Use Case
- Der neue Code wird automatisch auf den Server übertragen.

## Tips
- Frequently used in software development for data transmission (Daten übertragen) or code deployment.
- The prefix "über-" is inseparable in this context, so the past participle is "übertragen" without "ge-".

## Caution
- Do not confuse with "überweisen" (to transfer money). Use "übertragen" for data/tasks, and "überweisen" for bank transfers.
