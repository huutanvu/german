# erstellen

## Meanings
- to create
- to build / construct
- to compile / generate
- to draw up / prepare

## Grammar
- Part of speech: Verb (weak, regular)
- Perfekt: hat erstellt
- Conjugation:
  - ich erstelle
  - du erstellst
  - er/sie/es erstellt
  - wir erstellen
  - ihr erstellt
  - sie/Sie erstellen

## Daily Use Case
- Ich muss eine Liste für den Einkauf erstellen.

## Professional Use Case
- Wir erstellen jede Woche einen neuen Build der Anwendung.

## Tips
- Very common in IT for creating files, reports, tickets, or code.
- Often used with nouns like "Bericht" (report), "Dokumentation" (documentation), "Nutzerkonto" (user account).

## Caution
- Do not confuse with "darstellen" (to represent/portray) or "vorstellen" (to introduce/imagine).
